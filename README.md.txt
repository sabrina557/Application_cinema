# Application Cinéma

## Description
Cette mini-application présente une collection de films sous forme de **cartes stylées**. Chaque carte contient :
- Image du film
- Titre
- Description
- Évaluation par étoiles
- Bouton "Regarder la bande-annonce"

L'application est **responsive**, moderne et prête à être déployée sur GitHub Pages.

## Prompt IA utilisé
"Créer une application cinéma avec 12 films. Chaque film doit avoir une image, un titre, une description, une évaluation par étoiles (max 5) et un bouton pour regarder la bande-annonce. Les cartes doivent être centrées, stylées et disposées en grille de 4 cartes par ligne. Ajouter un titre moderne et animé pour la page."

## Instructions pour exécution
1. Télécharger ou cloner le projet
2. Ouvrir `index.html` dans un navigateur
3. Cliquer sur les boutons "Regarder la bande-annonce" pour ouvrir les trailers
